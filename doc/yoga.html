无线网络

apt install wireless-tools wpasupplicant

ifconfig wlp1s0 up
wpa_supplicant -i wlp1s0 -c /etc/wpa_supplicant/wpa_supplicant.conf &
dhclient wlp1s0

wpa_supplicant.conf==========
network={
ssid=”WiFi-SSID”
psk=”WiFi-passwd”
priority=1

}

声音输出

apt install alsa-utils
如果安装后所有声道默认是静音的，那么必须手动解除。执行 alsamixer ，下方标有 MM 的声道是静音的，而标有 00 的通道已经启用。
使用左右方向键，选中 Master 和 PCM 声道。按下 m 键解除静音。使用上下方向键增减音量，直到增益值为0。该值显示在左上方 Item: 字段后。过高的增益值会导致声音失真。
接下来，测试声卡是否工作： speaker-test -c 2
用-D参数来指定输出的设备： speaker-test -D default:PCH -c 2

系统待机

查看内核支持方式：cat /sys/power/state
挂起是挂起到内存：echo “mem” > /sys/power/state
休眠是挂起到SWAP：echo “disk” > /sys/power/stat

挂起：停用除了CPU RAM之外的外围设备并进入最低耗电模式。
休眠：将内存数据写入swap分区然后关机，下次开机直接从swap分区读取数据到内存。
